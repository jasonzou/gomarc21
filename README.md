# gomarc21

Another golang implementation of MARC 21.

## Functionality

- parse marc21 files
- convert marc21 into mrk format
- convert marc21 into marc21 xml format
- convert marc21 into marc21 json format
- validate marc21 json against avarm marc21 schema
- authority records
